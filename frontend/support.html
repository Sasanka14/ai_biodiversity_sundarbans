Common Issues & Solutions

Simulation not running

Make sure the backend server is running on localhost:3005. Start it with: uvicorn backend.app.main:app --port 3005

"Connection Error" message

Check your network connection and verify the API server is accessible. If using a firewall, ensure port 3005 is open for local connections.

Results seem incorrect

Verify that all 10 climate parameters are set within valid ranges. The model performs best when inputs reflect realistic Sundarbans climate conditions (20-35°C temperature, 40-95% humidity).

Export not working

Ensure you have run a simulation first before attempting to export. The export button generates a JSON file with your most recent simulation results.
