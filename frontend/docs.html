Complete guide to using the Sundarbans Biodiversity AI platform

Getting Started

The Sundarbans Biodiversity AI platform helps researchers and conservationists predict species richness changes based on climate variables. Our machine learning model analyzes temperature and humidity patterns to forecast ecosystem health.

Quick Start

  1. Navigate to the Simulator page
  2. Configure climate parameters using the sliders
  3. Click "Run Simulation" to generate predictions
  4. Review the risk assessment and AI interpretation
  5. Export results for further analysis

Climate Parameters

The simulator accepts 10 climate parameters organized into three categories:

Parameter Description Range
Air Temperature Current ambient temperature 20°C - 35°C
Relative Humidity Current moisture percentage 40% - 95%
Temp Lag 3M Temperature from 3 months ago 20°C - 35°C
Humidity Lag 3M Humidity from 3 months ago 40% - 95%
T2M Lag 1M/3M/6M NASA POWER 2m air temperature 20°C - 35°C
RH2M Lag 1M/3M/6M NASA POWER 2m relative humidity 40% - 95%

Understanding Results

Key Metrics

  • Baseline Richness: Reference species count under normal conditions
  • Predicted Richness: Forecasted species count based on input parameters
  • Change (Δ): Difference between predicted and baseline values

Risk Levels

  • High Risk (Δ ≤ -15): Severe ecosystem stress requiring immediate intervention
  • Moderate Risk (Δ ≤ -5): Notable pressure on species populations
  • Low Risk (Δ > -5): Ecosystem within healthy parameters

Data Sources

Our model is trained on multiple data sources:

  • NASA POWER: Satellite-derived meteorological data
  • Wildlife Observations: Field surveys from the Sundarbans region
  • Global Mangrove Watch: Mangrove extent and change data
  • Historical Climate Records: Multi-decadal temperature and humidity patterns
