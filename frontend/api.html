API Reference

RESTful API for biodiversity predictions and ecosystem analysis

Base URL:
http://localhost:3005
POST /simulate

Run a biodiversity simulation based on climate parameters. Returns predicted species richness, baseline comparison, and risk assessment.

Request Body
Parameter Type Description
air_temperature required float Current air temperature (20-35Â°C)
humidity required float Relative humidity percentage (40-95%)
temp_lag_3 required float Temperature from 3 months ago
hum_lag_3 required float Humidity from 3 months ago
T2M_lag_1, T2M_lag_3, T2M_lag_6 required float NASA POWER 2m temperature (1/3/6 month lag)
RH2M_lag_1, RH2M_lag_3, RH2M_lag_6 required float NASA POWER 2m humidity (1/3/6 month lag)
Example Request
POST /simulate Content-Type: application/json { "air_temperature": 28.5, "humidity": 75, "temp_lag_3": 27.0, "hum_lag_3": 72, "T2M_lag_1": 28.0, "RH2M_lag_1": 70, "T2M_lag_3": 27.0, "RH2M_lag_3": 72, "T2M_lag_6": 26.0, "RH2M_lag_6": 75 }
Response
{ "baseline": 180.0, "prediction": 165.5, "delta": -14.5, "risk_level": "Moderate Risk" }
Response Fields
Field Type Description
baseline float Reference species richness value
prediction float Predicted species richness under given conditions
delta float Change from baseline (prediction - baseline)
risk_level string "High Risk", "Moderate Risk", or "Low Risk"
